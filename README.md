## How to run system
1. Download games.xls from this Kaggle page and insert into the data/ folder.
[https://www.kaggle.com/datasets/fronkongames/steam-games-dataset](https://www.kaggle.com/datasets/fronkongames/steam-games-dataset)
2. Run build_model.py.
3. Run app.py.
